THE USER GUIDE
 -- to install packages : open terminal in root and run command: npm install --
1-START SERVER
    Open terminal from root
    To start the server, you need to run the command: npm start
    enter protocol, ip address, port 
2-START CLIENT
    Open new terminal from root
    To start client, you need to run this command: node client.js
    enter protocol, ip address, port the same as you entered in server
3-SEND MESSAGE IN CLIENT TERMINAL
4-WRITE CLOSE TO SHUT DOWN IN CLIENT TERMINAL
